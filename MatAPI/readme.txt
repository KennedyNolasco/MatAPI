
O teste1 faz o teste sobre as operações de matriz com matriz, que são a multiplicação de duas matrizes, soma de duas matrizes e subtração de duas matrizes. 
O teste2 faz o teste sobre as operações de matriz com escalar, que são a multiplicação de matriz por escalar, soma de matriz por escalar e subtração de matriz por escalar.
O teste3 faz o teste sobre as operações com matrizes, como achar a transposta de uma matriz, achar o determinante de uma matriz e achar a matriz inversa de uma matriz.
O teste 4 faz o teste sobre a integral com a função func definida na main, o intervalo do teste é de 0 até 10, e dividiu esse intervalo em 1000 intervalos menores para o cálculo da integral numérica, para realizar o teste nós integramos a função x^2, mas qualquer função pode ser definida na main para ser integrada.

Para rodar o programa basta usar o comando "make MatAPI", e em seguida usar o comando "make run", caso queira utilizar os testes basta usar o comando "make run < teste1.txt" por exemplo, pode substituir teste1.txt por qualquer um dos 4 testes.
